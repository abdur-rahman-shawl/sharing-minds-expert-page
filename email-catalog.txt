                    SHARINGMINDS EMAIL CATALOG

================================================================================
EMAIL #1: OTP Verification Email
================================================================================

TRIGGER:
  - When: User requests email verification during login/signup
  - API Endpoint: POST /api/auth/send-otp
  - Source File: lib/otp.ts (sendVerificationOtp function)

RECIPIENT: User's email address

SUBJECT: "Your Verification Code"

FROM: "Sharing Minds" <GMAIL_APP_USER>

CONTENT:
  ┌─────────────────────────────────────────────────────────────────────┐
  │                    Email Verification                              │
  │                                                                     │
  │  Thank you for signing up. Please use the following code to        │
  │  verify your email address:                                        │
  │                                                                     │
  │                    [ 6-DIGIT OTP CODE ]                            │
  │                                                                     │
  │  This code will expire in 10 minutes.                              │
  └─────────────────────────────────────────────────────────────────────┘

EXPIRY: 10 minutes from sending

DATABASE: Stores OTP in email_verifications table


================================================================================
EMAIL #2: Application Received Email
================================================================================

TRIGGER:
  - When: Mentor successfully submits registration form
  - API Endpoint: POST /api/mentors/apply
  - Source File: lib/emails.ts (sendApplicationReceivedEmail function)

RECIPIENT: Mentor's email address

SUBJECT: "We've Received Your Sharing Minds Application!"

FROM: "Sharing Minds" <GMAIL_APP_USER>

CONTENT:
  ┌─────────────────────────────────────────────────────────────────────┐
  │        Thank You for Applying to Sharing Minds!                    │
  │                                                                     │
  │  Hi [NAME],                                                        │
  │                                                                     │
  │  We're thrilled you want to join our community of mentors. This    │
  │  email confirms we've successfully received your application.      │
  │                                                                     │
  │  Our team will carefully review your profile. We'll get back to    │
  │  you within 5-7 business days with the next steps.                 │
  │                                                                     │
  │  In the meantime, feel free to explore our website and learn       │
  │  more about our mission.                                           │
  │                                                                     │
  │  Best regards,                                                     │
  │  The Sharing Minds Team                                            │
  └─────────────────────────────────────────────────────────────────────┘

TIMELINE: Sent immediately after successful form submission

EMAIL #3: Contact Form Submission (Internal)
================================================================================

TRIGGER:
  - When: User submits the contact form on /contact page
  - API Endpoint: POST /api/contact
  - Source File: lib/emails.ts (sendContactSubmissionEmail function)

RECIPIENTS (Internal Team):
  - community@sharingminds.in
  - support@sharingminds.in  
  - abdur.rahman.shawl@gmail.com

SUBJECT: "New Contact Enquiry: [USER'S SUBJECT]"

FROM: "Sharing Minds" <GMAIL_APP_USER>

CONTENT:
  ┌─────────────────────────────────────────────────────────────────────┐
  │              New Contact Enquiry                                   │
  │                                                                     │
  │  Name: [SENDER'S NAME]                                             │
  │  Email: [SENDER'S EMAIL]                                           │
  │  Subject: [SUBJECT LINE]                                           │
  │                                                                     │
  │  ┌─────────────────────────────────────────────────────────────┐   │
  │  │ Message                                                      │   │
  │  │ [FULL MESSAGE CONTENT]                                       │   │
  │  └─────────────────────────────────────────────────────────────┘   │
  └─────────────────────────────────────────────────────────────────────┘

NOTE: This is an internal notification email - sent TO the team, not the user.
      User's message content is HTML-escaped for security.


================================================================================
                    EMAIL CONFIGURATION
================================================================================

SMTP Provider: Gmail
Environment Variables Required:
  - GMAIL_APP_USER     (Gmail address for sending)
  - GMAIL_APP_PASSWORD (Gmail App Password, not regular password)

Source Files:
  - lib/emails.ts   - Application emails (confirmation, contact)
  - lib/otp.ts      - OTP verification emails
